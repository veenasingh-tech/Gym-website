# 🏋️‍♀️ Gym & Fitness Landing Page

Welcome to the **Gym Website** project!
 This is a modern, single-page landing page designed to provide a seamless user experience for fitness enthusiasts.

## 🚀 Live Demo
You can view the live website here: 
https://nimble-pasca-de6e96.netlify.app/

## ✨ Key Features
* **Modern UI/UX:**
 Clean and professional look with high-quality fitness imagery.
* **Fully Responsive:**
 Works perfectly on Mobile, Tablet, and Desktop screens.
* **Navigation Menu:** 
Smooth scrolling to different sections (Home, About, Plans, Contact).
* **Membership Plans:** 
Clearly displayed pricing and features table.
* **Interactive Elements:** 
Hover effects and buttons for better engagement.

## 🛠️ Tech Stack
* **HTML5:** Semantic structure for better SEO and accessibility.
* **CSS3:** Custom styling using Flexbox and Grid for a responsive layout.
* **JavaScript (Optional):** Basic interactivity for menus and scrolling.
